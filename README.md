# Hotel Booking Management System

## Use Case 1: Application Entry & Welcome Message

### Overview
This program represents the starting point of the Hotel Booking Management System.  
It demonstrates how the application begins execution using the Java `main()` method.

At this stage, the system only performs basic initialization and displays a welcome message.

---

## Objective
The purpose of this use case is to:

- Establish the main entry point of the application
- Ensure the program launches correctly
- Display a welcome message to the user
- Confirm that the system has initialized successfully

No business logic, user input handling, or database operations are implemented in this stage.

---

## Technologies Used

- Java
- JVM (Java Virtual Machine)
- Command Line Interface

## How to Compile and Run

### Step 1: Compile the Program

Open terminal or command prompt and run:


javac UseCase1HotelBookingApp.java


This command compiles the Java source file and generates a `.class` file.

---

### Step 2: Run the Program

Execute the program using:


java UseCase1HotelBookingApp
